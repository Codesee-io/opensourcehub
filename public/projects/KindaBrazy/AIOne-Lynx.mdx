<Overview>

AIOne Lynx is an open-source AI management software that lets you install, manage and launch all AI web-based user interfaces (WebUI) in one place.

AIOne Lynx is developed in Electron, TypeScript, and React. It aims to be the ultimate cross-platform software to keep all your available AI in one place with a clean, simple and modern UI.

Manage WebUI:
- Clone from GitHub
- Locate pre-installed WebUI
- Edit command line arguments
- Filter command line arguments
- In-app console
- In-app WebUI browser
- Detect and autolaunch UI

Currently supported WebUI:
- AUTOMATIC1111	stable-diffusion-webui
- LSHQQYTIGER     stable-diffusion-webui-directml
- OOBABOOGA	    Text generation web UI

</Overview>
