<Overview>

A classic console/pc emulator for iOS and tvOS. Provenance contains many upstream cores with custom macOS/iOS/tvOS wrapping code and core code modifications where necessary.

It has a legacy UIKit and new SwiftUI frontend, database agnostic interface (currently on Realm.swift) and various native extensions such as Spotlight, SiriKit, Shortcuts, iOS widgets etc.

</Overview>
