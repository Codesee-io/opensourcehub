<Overview>

Octokit-lite is an app powered by OctokitJS API by GitHub & Supabase for GitHub users to perform operations on multiple repositories efficiently. Available operations:

- Unfork: delete forked/created repos
- Unlabel: remove label(s) from all issues of selected repos
- TopicSpace: add topic(s) to selected repos

</Overview>
